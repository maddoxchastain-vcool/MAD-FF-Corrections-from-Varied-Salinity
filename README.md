# Correcting Moisture and Density (MAD) and Formation Factor (FF) using Varied Salinity Values in Offshore Freshened Groundwater Systems #
Author: Maddox Chastain

### Introdruction ###
This project is able to be carried out thanks to the Summer Undergraduate Research Fellowship (SURF) awarded to me at the Colorado School of Mines for the summer of 2026. I'm doing this work to aid in the onshore analysis of data collected on NSF IODP Expedition 501, which set out during the summer of 2025 to study an offshore freshened groundwater system offshore Massachusetts, south of Martha's Vineyard.
